# Barreto Advocacia - Site Oficial

Landing page premium para o escritório **Barreto Advocacia**, desenvolvida com foco em alta conversão, design "Liquid Glass" e performance excepcional.

## 🚀 Tecnologias

- **Vite** (Build Tool)
- **React** (Frontend Framework)
- **TailwindCSS** (Estilização)
- **Lucide React** (Ícones)
- **Framer Motion** (Animações - opcional/planejado)

## 📦 Estrutura de Ativos

As imagens e logos estão organizadas em:
- `public/assets/branding/logo/`: Logos oficiais (Navbar e Rodapé).
- `public/assets/branding/favicon/`: Favicon oficial.
- `public/assets/images/hero/`: Backgrounds da seção Hero (Desktop e Mobile).
- `public/assets/images/team/`: Fotos da equipe jurídica.
- `public/assets/images/contact-options/`: Imagens das seções de reuniões.

## 🛠️ Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 🏗️ Deploy (Vercel)

O projeto está configurado para deploy imediato na Vercel:
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`
